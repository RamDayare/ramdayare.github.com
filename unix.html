Unix "chmod" Command..


The Unix chmod command is used to change the access mode of a file. The following is the chmod command syntax.

chmod who=permissions filename

Who
“who” is a list of letters that specifies whom you’re going to be giving permissions to.
These permissions are specified in any order.
u   - indicates owner of the file
g   - indicates group the file belongs to.
o   - indicates other users
a   - all of the above

Permissions
Permissions are used to give read,write,delete and execute acess of files to different users or group.Below are the 3 permissions which user can assign on file :
r  - Permission to read the file.
w - Permission to write or delete the file.
x  - Permission to execute the file.

The ls -l command is used to find out the permissions on the files.
Example
$ ls -l Test.txt            
-rw-rw-r-- 1 dev dev 0 2012-02-18 01:06 Test.txt

If user wants that file Test.txt needs to be readable only to everyone then he can change the permission by using below command
$ chmod a=r Test.txt
-r--r--r-- 1 dev dev 0 2012-02-18 01:06 Test.txt

If user wants that file Test.txt should have read and write access to everyone then he can change the permission by using below command
$ chmod a=rw Test.txt
-rw-rw-rw- 1 dev dev 0 2012-02-18 01:06 Test.txt

If user wants that file Test.txt should have all access to everyone then he can change the permission by using below command
$ chmod a=rwx Test.txt         
-rwxrwxrwx 1 dev dev 0 2012-02-18 01:06 Test.txt


Instead of  'a'  which indicates all the groups,owners and other users,we can also specify particular user or group to whome we can assign the different access permission as required.
The below command gives read write access to groups only.The owner and other users have only read permission.
$chmod g=rw Test.txt          
-r--rw-r-- 1 dev dev 0 2012-02-18 01:06 Test.txt
