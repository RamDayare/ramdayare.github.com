Reverting in Git..

As a human being we are making lot of mistakes, but what we can do if we are making mistakes while using github. Let say we have added some changes in files or tree structures which could be in commited or uncommited state and we want to revert those changes so....

How can we revert changes in Git?

We have reset , checkout and revert commands in Git which provides us facility to reverting those changes

1. Reverting Uncommitted changes

Reset command: Used to undo the changes on tree or multiple file level

Syntax : $ git reset --hard HEAD

If you are doing some changes in file or tree level and these uncommited changes you don't
want to be pushed in repository then you can use 'git reset --hard HEAD' command in Git
which provides way to revert the mistakes.

E.g.
~/Root$ Touch Test
~/Root$ ls
File1 File2 Test
~/Root$ git add .
~/Root$ git reset --hard HEAD 'Removes Test file which is added currently '
~/Root$ ls
File1 File2

Checkout command: Used to undo the changes on file level

Syntax : $ git checkout -- filename
              $ git checkout .

If you want to reset changes made in single file then use "$git checkout -- filename" and for multiple files use "$git checkout ."

E.g.
~/Root$ ls
File1 File2 Test
~/Root$ cat Test
This is first line
~/Root$ cat >>Test
This is Second line
~/Root$ cat Test
This is First line
This is Second line
~/Root$ git checkout -- Test     ' Revert the last changes made in Test file '
~/Root$ cat Test
This is First line

2. Reverting committed changes

Revert command : Used when changes are commited and we want to revert the changes

Syntax : $ git revert HEAD

If we want to revert changes in files which have commited recently then we can use revert command to revert those changes.This will create a new commit which undoes the change in HEAD.You will be given a chance to edit the commit message for the new commit.
 

E.g.
~/Root$ ls
File1 File2 Test
~/Root$ cat >>Test
This is  Second line
~/Root$ cat Test
This is First line
This is  Second line
~/Root$ git add .
~/Root$ git commit  Test -m 'Commited file'
~/Root$ git revert HEAD                 ' Revert the changes comiited in Test file'
~/Root$ cat Test
This is First line
